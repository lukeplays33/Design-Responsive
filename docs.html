Functions.

setSize()

>

setSize(Width,Height);

Sets the values for the screen width and height, used to calculate values for each CSS property.

Input Accepts
Width Number
Height Number

getWidth() and getHeight()

>

getWidth();

getHeight();

Returns the calculated value of the screen width or height set by the setSize() function.

Function Returns
getWidth() Number
getHeight() Number

getResizedElements()

>

getResizedElements();

Returns all elements that have been resized.

Function Returns
getResizedElements() Array

resizeElement()

>

resizeElement(element,properties);

Resizes the given element's CSS properties.

Input Accepts
Element HTMLElement
Properties Array of CSS property names

resizeAllElements()

>

resizeAllElements(elements,properties);

Resizes all element's inside of the body.

Input Accepts
Elements Array of HTMLElement
Properties Array of CSS property names

FAQ.

Longhand and Shorthand support.

>

Longhand and Shorthand names are supported but it's recommended to never use both of them at the same time.

E.g. if your css has both margin and margin-left never use margin as this will not work.

Insted use margin-left,margin-right,margin-top and margin-bottom if both are in use.

Reloading an Iframe.

>

If you are using an iframe you may have noticed that the SRC is loaded twice.

This is to make sure that all webpages inside of the Iframe are rendered correctly.

Bacause some sites center specific elements, the SRC is reloaded to make sure everything is aligned correctly and does not stick to the old iframe size.
